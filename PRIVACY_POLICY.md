# Privacy Policy for Postify Chrome Extension

**Last updated:** [Date]

## Introduction

Postify ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our Chrome extension.

## Information We Collect

### 1. Gmail Data
- **Certificate Emails**: We scan your Gmail for emails containing certificates, achievements, and course completions
- **Email Content**: We extract relevant information from certificate emails including:
  - Subject lines
  - Email body content
  - Sender information
  - Date received

### 2. User Configuration
- **API Keys**: OpenAI API key for AI post generation
- **Social Media Tokens**: Twitter Bearer token and LinkedIn credentials
- **Settings**: User preferences for tone, platforms, and scanning intervals

### 3. Generated Content
- **AI-Generated Posts**: Social media posts created using OpenAI's API
- **Draft Posts**: Posts saved by users for later use

## How We Use Your Information

### 1. Certificate Processing
- Scan Gmail for certificate-related emails
- Extract certificate details and skills
- Store certificate information locally for post generation

### 2. AI Post Generation
- Send certificate information to OpenAI API
- Generate social media posts based on your preferences
- Return generated content to your device

### 3. Social Media Posting
- Use your social media tokens to post content
- Send posts directly to Twitter and LinkedIn
- Retrieve post URLs for your reference

## Data Storage and Security

### 1. Local Storage
- All data is stored locally on your device using Chrome's storage API
- No data is transmitted to our servers
- Data persists only on your device and can be cleared at any time

### 2. API Communications
- Direct communication with third-party APIs:
  - Google Gmail API (for email scanning)
  - OpenAI API (for post generation)
  - Twitter API (for posting)
  - LinkedIn API (for posting)

### 3. Security Measures
- API keys and tokens are stored securely in Chrome's encrypted storage
- All API communications use HTTPS encryption
- No sensitive data is logged or transmitted to our servers

## Your Rights and Choices

### 1. Data Access
- View all stored data through the extension's settings
- Export your data using the extension's export feature
- Clear all data at any time

### 2. Data Control
- Revoke Gmail access through Google Account settings
- Remove API keys and tokens from settings
- Delete individual certificates or posts

### 3. Opt-Out
- Uninstall the extension to remove all data
- Disable automatic scanning in settings
- Stop using the extension at any time

## Data Retention

### 1. Local Data
- Data is retained until you:
  - Clear it manually
  - Uninstall the extension
  - Clear browser data

### 2. Third-Party Data
- Gmail data: Subject to Google's retention policies
- OpenAI data: Subject to OpenAI's retention policies
- Social media data: Subject to respective platform policies

## Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect personal information from children under 13.
